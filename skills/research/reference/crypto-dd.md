# Crypto Due Diligence — Reference

## Crypto-Specific Layers (on top of base 5-layer stack)

Crypto adds unique dimensions:
1. **24/7 market** — no market hours, gaps happen differently
2. **On-chain data** — transparent flow data unavailable in equities
3. **Narrative cycles** — crypto moves on themes (DeFi summer, AI tokens, L2s)
4. **Correlation to BTC** — most alts follow BTC; only trade alts when BTC is stable/bullish

## Scanning Criteria

| Criterion | Requirement |
|-----------|-------------|
| Market cap | > $100M (avoid micro-caps) |
| 24h volume | > $10M |
| Listed on major exchange | Coinbase, Binance, Kraken |
| BTC correlation check | BTC not in freefall (> SMA20 on daily) |
| Narrative alignment | Fits current market theme |

## Candidate Universe

**Large cap (core):** BTC, ETH, SOL, AVAX, LINK, DOT, MATIC, NEAR
**Mid cap (higher risk/reward):** ARB, OP, INJ, SUI, APT, SEI, TIA, JUP
**Sector plays:** AI tokens, DeFi, L2s, RWA — rotate based on narrative

## Crypto-Specific Analysis

### On-Chain Signals (when available)
- Exchange inflows/outflows (large outflows = accumulation = bullish)
- Whale wallet movements (> $1M transfers)
- Active addresses trend (growing = adoption)
- TVL changes for DeFi tokens

### Narrative & Catalyst Assessment
- Is there a protocol upgrade coming? (ETH merge, Solana Firedancer)
- New partnership or integration announced?
- Regulatory news (positive or negative)?
- Is this narrative fresh or exhausted?

### BTC Dominance Check
- BTC dominance rising → alts underperform, stick to BTC
- BTC dominance falling → alt season, rotate to high-beta alts
- BTC ranging → selective alt plays OK

## Scoring Adjustments for Crypto

| Factor | Bonus/Penalty |
|--------|---------------|
| BTC above SMA20 + SMA50 | +10 |
| BTC below SMA20 | -15 (alt risk high) |
| Fresh narrative catalyst | +15 |
| Stale narrative (> 1 week) | -10 |
| Exchange outflows increasing | +10 |
| Exchange inflows spiking | -10 (selling pressure) |
| Token unlock within 7 days | -15 (supply pressure) |
| Protocol upgrade within 30 days | +10 |

## Entry Rules (Crypto-Specific)

- **Avoid entries during BTC volatility spikes** (wait for BTC to settle)
- **Limit orders only** — crypto spreads can be wide
- **Size smaller than equities** — 0.5-1% risk per trade (higher volatility)
- **Check funding rates** (for perpetuals) — extreme funding = crowded trade
- **Weekend entries OK** but expect lower liquidity

## Exit Framework (Crypto-Specific)

| Type | Condition |
|------|-----------|
| Take-profit | 3:1 R/R minimum (crypto moves are larger) |
| Stop-loss | Below key support or 2× ATR |
| Trailing | After 2R: trail by 1.5× ATR |
| BTC breakdown | If BTC breaks below SMA50, exit all alts |
| Narrative death | If the theme is over (regulatory kill, hack), exit immediately |

## Red Flags (Auto-Reject)

- BTC below SMA50 (bear market for alts)
- Token unlock > 5% of supply within 7 days
- Market cap < $100M (manipulation risk)
- Only listed on DEX (liquidity/rug risk)
- Team anonymous + no audit (scam risk)
- Narrative is > 2 weeks old with no new catalyst
- Funding rate > 0.1% (crowded long, squeeze risk)
